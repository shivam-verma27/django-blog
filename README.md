# Django Blog Platform

A full-stack blog application built using Django that allows users to register, log in, create posts, categorize content, comment on posts, like posts, search articles, and manage their profiles. The application is deployed on Render using PostgreSQL.

## Installation

Clone the repository

```bash
git clone https://github.com/shivam-verma27/django-blog.git
```

Go to the project folder

```bash
cd django-blog
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux/macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Apply migrations

```bash
python manage.py migrate
```

Create a superuser

```bash
python manage.py createsuperuser
```

Run the development server

```bash
python manage.py runserver
```
